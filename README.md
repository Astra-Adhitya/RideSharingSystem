# 🚖 Ride Sharing System

## Overview
A modular ride-sharing system built using C++ with object-oriented programming.

## Architecture
Frontend (Web UI)
↓
Flask Backend
↓
C++ Core Engine
↓
CSV Data Storage

## Features
- Driver and Passenger management
- Ride booking
- Fare calculation using polymorphism
- CSV-based data storage

## Structure
- cpp/ → core logic
- web/ → dashboard
- data/ → storage
- docs/ → diagrams

## Status
Project setup completed. Core engine in development.
